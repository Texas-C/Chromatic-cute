# ColorSortGame
1. How to make
	1). install libsdl2-dev
		apt install libsdl2-dev #ubuntu

	2). make / clean
		make / make clean

2. RUn
	./MainWin

3. config
	Change XD.txt

	example:

	//XD.txt:
	
		0	// left bar's RGB value
		64
		125

		0	// right bar's RGB value
		64
		63

		20	//length
	// XD.txt end
